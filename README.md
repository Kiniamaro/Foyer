# Foyer
Self contained modular start page webserver

---

## Setup
```bash
# setup virutalenv
$ virtualenv venv
$ . venv/bin/activate

# install python dependencies
$ pip install -r requirements.txt

# install js dependencies
$ npm install

# compile js code
$ npm run webpack
```

## Usage
```bash
# run the app
$ python foyer.py

# Congrats! the app is now available on http://localhost:9001/

```
